    <dl class="hwprdeflist" data-hederis-type="hwprdeflist" id="p6yYgFsHr"><dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p81JoW5AS">HED Address</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lidsi4sm2d"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pFun2OiII">An address, for example in a letter, on a copyright page, etc.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pYGrrRKhn">HED Attribution</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liedxpcf9j"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pftZagUiX">The attribution to a quote.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="plRG09Nvs">HED Attribution (contd)</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lisL0C7CEV"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pS6lYVji9">Continuation paragraphs for a quote attribution. An attribution can have multiple continuation paragraphs.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p2hfrd1Ei">HED Author name</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lio9b5lumU"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pHYwxT5TQ">The name of an author. This could appear in a variety of places: on the main titlepage, following a preface title or chapter title, etc.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pveY2eVy6">HED Box type</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li3xcggvrD"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p6gEygl9C">Some books include multiple types of boxes and sidebars, that appear multiple times within a book. For example, a book might have a type of box called &#8220;Quick Tips&#8221;, that contains different tips that correspond to the current book topic. This style would be used to tag that Type heading, and is different from the main box heading.</p><img data-hederis-type="hblkimg" class="hblkimg" id="px3Guzn81" src="/images/boxtype.png" data-img-src="boxtype.png"/>
    </dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pR0UkWtyp">HED Caption</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lib6VvAXso"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="ppJW61b6w">A caption, for example to accompany an image or to follow a table.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pGl6RMojC">HED Chap/Sect/Wrapper number</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liwu6Nw9bf"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pXtcybOvU">Number labels for any book section or wrapper, e.g., &#8220;Chapter 1&#8221;, or &#8220;Example 12&#8221;.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pmXi3w2AA">HED Chapter/Section title</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liho1DrNKV"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pI116EBWZ">Titles of main book sections, like chapters, prefaces, appendixes, parts, and so on.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p85wJjNQu">HED Closing</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li3c08lIh1"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="phT1aM0eN">The closing line of a letter, e.g., &#8220;Sincerely,&#8221;.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pXGMwSfmr">HED Code paragraph</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liFrHebPoN"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="piev035AB">A block of computer code.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pcFN7VKD1">HED Contributor</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lizbExpcyZ"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pKDrN0B3K">A book or section contributor, other than the author. For example, the name of an illustrator or editor that might appear on the titlepage or at the beginning of a chapter.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pd2FsSGm4">HED Dateline</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lil5YhvERC"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pcqcjJJBU">A dateline.</p></dd>

    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liWheVPvbw"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pTcQCjVdT">The definition paragraph in a set of term/definition pairs. This paragraph is an example of the definition paragraph that corresponds to the term above.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="ppaSwa6Ld">HED Definition definition (contd)</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liixR0b1a8"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pjCy0VAS7">A continuation paragraph for a multi-paragraph definition, as defined above.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="puXkCNeHX">HED Definition term</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liWlqRRZbk"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="paILYPCvA">A term to be defined, often followed by a <span class="Emphasis" id="pQwB3cAfz"><em class="hspanem" data-hederis-type="hspanem" id="pyX2U6QUS">HED Definition definition</em></span> paragraph.</p></dd>

    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liEmW8viYd"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pj9Ehzgn5">A paragraph in an endnote.</p></dd>

Also often used as a heading preceding a letter body, e.g., &#8220;CONFIDENTIAL&#8221;.</p></dd>

For multi-level lists, use the <span class="Emphasis" id="pu6DCmKnk"><em class="hspanem" data-hederis-type="hspanem" id="pAXRyLa7E">HED BOX Num List</em></span> wrapper, nesting multiple levels of list wrappers within it, like this:</p><img data-hederis-type="hblkimg" class="hblkimg" id="plEOZ5Ljq" src="/images/nestnumlist.png" data-img-src="nestnumlist.png"/>

    <section class="hwprsubsection" data-hederis-type="hwprsubsection" id="pTeG5hIDW" data-type="subsection" title="Inline Styles"><h1 data-hederis-type="hblktitle" class="hblktitle" id="ppThk9QO7">Inline Styles</h1>
    <p class="hblkp" data-hederis-type="hblkp" id="pMbQZCWNd">Inline styles are styles that you apply just to a few words or characters within a paragraph, without affecting the rest of the text in the paragraph. Hederis supports most of Word&#8217;s built-in character formatting tools (like italics, bold, underline, etc.), so you don&#8217;t need to use inline styles for everything. However, there are a few cases where inline styles come in handy:</p>
    <dl class="hwprdeflist" data-hederis-type="hwprdeflist" id="pfbMyHndp"><dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pT1tPRnEv">HED SPAN Small Caps Text</dt>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pPJS4OrI3">HED SPAN Emphasis Text</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liZ3kDm2hb"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pVBBWShDe">Text that should be emphasized both visually and while being read aloud. This style is typically represented as italics. (In modern web development, &#8220;emphasis&#8221; has replaced &#8220;italic&#8221; as a code to add emphasis to your text.)</p><p class="hblkdefinitioncont" data-hederis-type="hblkdefinitioncont" id="pa9Y24MbD">Note that you can use Word&#8217;s built-in Italic formatting button, and this style will automatically be applied when the .docx file is converted in Hederis.</p>
    </dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pLG3genDi">HED SPAN Strong Text</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liuYMi7xXz"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p9CZxgfaE">This style is typically used for bold text. (In modern web development, &#8220;strong&#8221; has replaced &#8220;boldas a code to add weight to your text.)</p><p class="hblkdefinitioncont" data-hederis-type="hblkdefinitioncont" id="pDRpdPk67">Note that you can use Word&#8217;s built-in bold formatting button, and this style will automatically be applied when the .docx file is converted in Hederis.</p>
    </dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pAAeAjbmx">HED SPAN Superscript Text</dt>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pVxLDedes">HED SPAN Subscript Text</dt>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pfuT4M0CY">HED SPAN Inline Link</dt>
