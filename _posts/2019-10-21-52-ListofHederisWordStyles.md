    <dl class="hwprdeflist" data-hederis-type="hwprdeflist" id="pyPOVSBDK"><dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pkNbLogEg">HED Address</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liNQxq88x9"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pmIiraf7A">An address, for example in a letter, on a copyright page, etc.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pmbon7Aeu">HED Attribution</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liirdtQB06"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p13Ll8Jsx">The attribution to a quote.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="ps4CmTqPl">HED Attribution (contd)</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liEdLc51MV"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="ptzQ9Pwwt">Continuation paragraphs for a quote attribution. An attribution can have multiple continuation paragraphs.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pJZqsttlP">HED Author name</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liOZvAmPNV"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pPIqqlRBC">The name of an author. This could appear in a variety of places: on the main titlepage, following a preface title or chapter title, etc.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pTluHMVNf">HED Box type</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lisBgDDsFS"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p6efljcxU">Some books include multiple types of boxes and sidebars, that appear multiple times within a book. For example, a book might have a type of box called &#8220;Quick Tips&#8221;, that contains different tips that correspond to the current book topic. This style would be used to tag that Type heading, and is different from the main box heading.</p><img data-hederis-type="hblkimg" class="hblkimg" id="pIglqaEHp" src="/images/boxtype.png" data-img-src="boxtype.png"/>
    </dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p9ThP6LEj">HED Caption</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li3pRLPLcd"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pUPNk6Dtd">A caption, for example to accompany an image or to follow a table.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pKSC7rsks">HED Chap/Sect/Wrapper number</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lilOM6DjeN"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="prmAZ6xgH">Number labels for any book section or wrapper, e.g., &#8220;Chapter 1&#8221;, or &#8220;Example 12&#8221;.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pCf8uxAAE">HED Chapter/Section title</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liSdNAjgz8"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pcAmuZS6o">Titles of main book sections, like chapters, prefaces, appendixes, parts, and so on.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p1lnwLSGK">HED Closing</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liJhnWfe7q"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pUFJp53Tp">The closing line of a letter, e.g., &#8220;Sincerely,&#8221;.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pJZYPYIkz">HED Code paragraph</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liEXaZr4XM"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p2aUhxpbp">A block of computer code.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p5MOrdM4e">HED Contributor</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liwGDZ0AXS"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pPFUXsfwo">A book or section contributor, other than the author. For example, the name of an illustrator or editor that might appear on the titlepage or at the beginning of a chapter.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pXOfUDuf7">HED Dateline</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liU2KNXLoL"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p48FUsWMU">A dateline.</p></dd>

    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liqzjAhMG1"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pzPBiV3EA">The definition paragraph in a set of term/definition pairs. This paragraph is an example of the definition paragraph that corresponds to the term above.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pQHsmauYf">HED Definition definition (contd)</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li0w9DasDx"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p6xGINMQY">A continuation paragraph for a multi-paragraph definition, as defined above.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pAEjCKdIJ">HED Definition term</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li3sCpNYSa"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="paicjW9V1">A term to be defined, often followed by a <span class="Emphasis" id="pXutl0Wrc"><em class="hspanem" data-hederis-type="hspanem" id="pXQ3QKen7">HED Definition definition</em></span> paragraph.</p></dd>

    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li34OAbE7E"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p8C7lvVPq">A paragraph in an endnote.</p></dd>

Also often used as a heading preceding a letter body, e.g., &#8220;CONFIDENTIAL&#8221;.</p></dd>

    <section class="hwprsubsection" data-hederis-type="hwprsubsection" id="pvpLV2Iv4" data-type="subsection" title="Inline Styles"><h1 data-hederis-type="hblktitle" class="hblktitle" id="pAiMdO9k8">Inline Styles</h1>
    <p class="hblkp" data-hederis-type="hblkp" id="pYIAuhCe8">Inline styles are styles that you apply just to a few words or characters within a paragraph, without affecting the rest of the text in the paragraph. Hederis supports most of Word&#8217;s built-in character formatting tools (like italics, bold, underline, etc.), so you don&#8217;t need to use inline styles for everything. However, there are a few cases where inline styles come in handy:</p>
    <dl class="hwprdeflist" data-hederis-type="hwprdeflist" id="p5bi7iLqI"><dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pDKE3Xllp">HED SPAN Small Caps Text</dt>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="prrfjH6hV">HED SPAN Emphasis Text</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liQXbeigde"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pDs32DKPK">Text that should be emphasized both visually and while being read aloud. This style is typically represented as italics. (In modern web development, &#8220;emphasis&#8221; has replaced &#8220;italic&#8221; as a code to add emphasis to your text.)</p><p class="hblkdefinitioncont" data-hederis-type="hblkdefinitioncont" id="ppXJ9F0Un">Note that you can use Word&#8217;s built-in Italic formatting button, and this style will automatically be applied when the .docx file is converted in Hederis.</p>
    </dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pmWdDHstT">HED SPAN Strong Text</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liagh41nTb"><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pja7Noexi">This style is typically used for bold text. (In modern web development, &#8220;strong&#8221; has replaced &#8220;boldas a code to add weight to your text.)</p><p class="hblkdefinitioncont" data-hederis-type="hblkdefinitioncont" id="pn0Mg0WBW">Note that you can use Word&#8217;s built-in bold formatting button, and this style will automatically be applied when the .docx file is converted in Hederis.</p>
    </dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p5UDCC3yB">HED SPAN Superscript Text</dt>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="ptjj1jqtd">HED SPAN Subscript Text</dt>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pWVLCzHC5">HED SPAN Inline Link</dt>
