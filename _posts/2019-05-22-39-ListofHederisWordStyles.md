    <section class="hwprsubsection" data-hederis-type="hwprsubsection" id="pS69KQNac" data-type="subsection" title="Paragraphs"><h1 data-hederis-type="hblktitle" class="hblktitle" id="p0dGplhKP">Paragraphs</h1>
    <dl class="hwprdef-list" data-hederis-type="hwprdef-list" id="p7bkuMMrG"><dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pK6MRdkdH">HED Address</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liqNNmAagC"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pS9i0TTMe">An address, for example in a letter, on a copyright page, etc.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pXfRNtbdP">HED Attribution</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li7NflZgri"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pwiT9n8vf">The attribution to a quote.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pOItr4PGO">HED Attribution (contd)</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lilIv9pYo7"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="p54uWX7Sk">Continuation paragraphs for a quote attribution. An attribution can have multiple continuation paragraphs.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pH5sDajee">HED Author name</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liFiWamMem"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="p6hpXgnQb">The name of an author. This could appear in a variety of places: on the main titlepage, following a preface title or chapter title, etc.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p3YLmmxHL">HED Box type</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lilnykieOE"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="p4Gy1TBIA">Some books include multiple types of boxes and sidebars, that appear multiple times within a book. For example, a book might have a type of box called &#8220;Quick Tips&#8221;, that contains different tips that correspond to the current book topic. This style would be used to tag that Type heading, and is different from the main box heading.</p><img data-hederis-type="hblkimg" class="hblkimg" id="pn9NvNUAZ" src="/images/boxtype.png"/>
    </dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pYEg8kX6l">HED Caption</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="livEKwDV8t"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="p91Gqa8e6">A caption, for example to accompany an image or to follow a table.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pRYKIzCh3">HED Chap/Sect/Wrapper number</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lieds0psQR"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pbqPnmwwT">Number labels for any book section or wrapper, e.g., &#8220;Chapter 1&#8221;, or &#8220;Example 12&#8221;.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pCXE6Exrm">HED Chapter/Section title</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liq35kMLif"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="psYCa6Mau">Titles of main book sections, like chapters, prefaces, appendixes, parts, and so on.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pZaNpWDJO">HED Closing</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li2032LhJB"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pGnu0KgbB">The closing line of a letter, e.g., &#8220;Sincerely,&#8221;.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pwpq15mjH">HED Code paragraph</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liQQpN9Y3M"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pkCT5L3bj">A block of computer code.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pvxS868CQ">HED Contributor</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lipFEGyChf"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pUlD55RGM">A book or section contributor, other than the author. For example, the name of an illustrator or editor that might appear on the titlepage or at the beginning of a chapter.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="po2azitpJ">HED Dateline</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liruDwx33M"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pxlqAc4WJ">A dateline.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pRP33tf1i">HED Definition definition</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lipuJYif6V"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="p51oRiOAf">The definition paragraph in a set of term/definition pairs. This paragraph is an example of the definition paragraph that corresponds to the term above.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pxLhVat9C">HED Definition definition (contd)</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li5d6MkJgn"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pRLJ8fGFi">A continuation paragraph for a multi-paragraph definition, as defined above.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="ptvvVgSeD">HED Definition term</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liV99QVxPt"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="ptrkmgGr6">A term to be defined, often followed by a <em>HED Definition </em><em>definition</em> paragraph.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p2aW91Boi">HED Dialogue paragraph</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liYUT6ZvJ0"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pI5g9Lbrh">Dialogue.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="ppA6ei7sa">HED Endnote paragraph</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liWctNrgds"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pVHWnGYCq">A paragraph in an endnote.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p9UIJO8qS">HED Footnote paragraph</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liQsMj9xbe"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pnyBPNqMW">A paragraph in a footnote.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pLXu6WZwW">HED Generic heading</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liBU8WBa3y"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pc6AVaxMK">A generic heading, often used in the middle of boxes or sidebars when you want to introduce a block of text without creating a formal sub-section. Also often used as a heading preceding a letter body, e.g., &#8220;CONFIDENTIAL&#8221;.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p8TVzbGKN">HED Generic subheading</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li42ZCIIaP"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="p5jDKpGzH">Similar to the generic heading, a second level of heading that can be used when you want to create a logical grouping of text without creating a formal sub-section.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pWO7JN9G8">HED Image holder</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liJvlL4L4I"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pbWQV12YQ">A placeholder for an image. See &#8221;A MICROSOFT WORD FIELD WAS REMOVED FROM THIS LOCATION&#8221; for more.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="poffrjGk8">HED Image source</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li0cStsQ9m"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="puUZrqUcO">The source of an image, e.g. &#8220;Courtesy of Getty Images&#8221;.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pgVfWiWwH">HED Imprint name</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liejs82sg9"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="piYs7ifIq">The name of your imprint, often included on a titlepage.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pHPUyhgYZ">HED Line space</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liLI1zT6bM"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pQvPXyM8I">A line space, often used to create a logical break in time or context.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pGRj9dxlh">HED List item (contd)</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li3G3GruMk"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="peDb9YbWl">A second paragraph in a multi-paragraph list item (can be used for both bulleted and numbered list items).</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="prg9S7nYy">HED List item &#8211; Bulleted</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lijqmfUED4"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pUR3mqZTC">A bulleted or unordered list item. For multi-level lists, use the <em>HED BOX Bullet List</em> wrapper, nesting multiple levels of list wrappers within it, like this:</p><img data-hederis-type="hblkimg" class="hblkimg" id="p0C3T9qHq" src="/images/nestbulllist.png"/>
    </dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pvrxnrffv">HED List item &#8211; Numbered</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lioDSy140d"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="p9Hz8HzMK">A numbered or ordered list item. For multi-level lists, use the <em>HED BOX </em><em>Num</em><em> List</em> wrapper, nesting multiple levels of list wrappers within it, like this:</p><img data-hederis-type="hblkimg" class="hblkimg" id="pyLQneXCZ" src="/images/nestnumlist.png"/>
    </dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p1u6znwE5">HED Location</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liWPYtTyMi"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="px2iB01v8">A location, for example the location of your office as it appears on the book titlepage, or the location in which a chapter is set.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pzCovrChf">HED Mark for Deletion</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lilqdzVkyN"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pJnQngCq4">A paragraph that should be deleted in the final output.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pv7IYy2bI">HED Ornamental space</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li60nSO8nU"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pN4WBcdBN">An ornamental break in the text, often used to create a logical break in time or context and containing a visual ornament.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pHgoYpYAD">HED Plain text paragraph</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li0E30L7F7"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pB8KEL6nI">This versatile style is used for plain text, for example in the main flow of a chapter, or a paragraph of text in an extract, or the body text of a letter. See &#8220;<a href="{% post_url 2019-05-22-13-ParagraphsWrappersandSections %}"><span class="Hyperlink">Paragraphs, Wrappers, and Sections</span></a>&#8221; for more about this.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pbWb4DzOV">HED Plain text paragraph (contd)</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li9gXGKFXd"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="prHusBzGm">If a paragraph is broken up by an image or other content, use this style for the continuing text of the paragraph.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pDAssXvvz">HED Postscript</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liqRVJIp0o"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="poOGFIvKv">A postscript, as in a letter or foreword, e.g., &#8220;P.S., don&#8217;t forget to buy milk!&#8221;.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p39jfviq5">HED Processing instruction</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liQZjvTjI5"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pn4ymlIOI">This special Hederis style is used to add extra instructions about how to transform the paragraphs and sections of your manuscript. See &#8220;<a href="{% post_url 2019-05-22-24-Addspeciallayoutinstructions %}"><span class="Hyperlink">Add special layout instructions</span></a>&#8221; to learn all about processing instructions.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p6a93nrvS">HED Publisher name</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liBPPXOB57"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pvbvy34Zq">The name of your publisher, for example appearing on a titlepage.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pb3B7B7MX">HED Quote paragraph</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liGSzuQlY1"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="p3L5nufTT">A quote, that you don&#8217;t necessarily want to appear inside an extract.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="px9Ob1L5d">HED Salutation</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liVMvxqe7X"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pBxBa1V0u">A salutation, for example in a letter or foreword, e.g. &#8220;Dear reader,&#8221;.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pqXccgNJ0">HED Signature</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="linHVYenOm"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pj3doEAuG">The signature of the author of a block of text, for example in a letter or foreword.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="ppDnR7QLt">HED Speaker name</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liitlX0Azs"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="p6YxoI8PS">A speaker in a block of dialogue.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="plTVYgZNi">HED Subtitle</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lix30Chusg"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pCM5xZWkQ">Subtitles for any type of section or wrapper. For example, could be used both for a chapter subtitle, and for a subtitle in a sidebar.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p6FOQbUiP">HED TOC entry &#8211; backmatter</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liJLIWhJU9"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pNAaXcjAP">A table of contents entry for a backmatter section.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pmBu1ijG1">HED TOC entry &#8211; chapter</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="lim5b17ibP"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="p9bBfu7ti">A table of contents entry for a chapter.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="ptO2tp852">HED TOC entry &#8211; frontmatter</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liIrOvxUv9"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pgRlJebQP">A table of contents entry for a frontmatter section.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p9S75svwg">HED TOC entry &#8211; part</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liCwzesuwk"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="phdjqM4cT">A table of contents entry for a part.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p1yBSTte7">HED Verse paragraph</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liOh2yW8tX"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pdBBzUCoO">A single line of verse.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pM3yoMH0Q">HED Wrapper title</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liuNLIoKjo"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pzqGhzqlM">Titles of wrappers and sub-sections. For example, for a sidebar title, extract title, sub-section title, etc.</p></dd>

    <section class="hwprsubsection" data-hederis-type="hwprsubsection" id="pZVAz5wKI" data-type="subsection" title="Inline Styles"><h1 data-hederis-type="hblktitle" class="hblktitle" id="pTJi9q9Y5">Inline Styles</h1>
    <p class="hblkp" data-hederis-type="hblkp" id="pOw16a51e">Inline styles are styles that you apply just to a few words or characters within a paragraph, without affecting the rest of the text in the paragraph. Hederis supports most of Word&#8217;s built-in character formatting tools (like italics, bold, underline, etc.), so you don&#8217;t need to use inline styles for everything. However, there are a few cases where inline styles come in handy:</p>
    <dl class="hwprdef-list" data-hederis-type="hwprdef-list" id="pUe9F8wwX"><dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pOI3Fpc9x">HED SPAN Loosen spacing</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liTOUlVzL4"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pOOdU3Qk9">Loosen the character spacing in a span of text in the PDF. See &#8220;<a href="{% post_url 2019-05-22-29-AdjustletterspacingPDF-only %}"><span class="Hyperlink">Adjust letter spacing (PDF-only)</span></a>&#8221; for more.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="psLmRD1y8">HED SPAN Tighten spacing</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liXdT1lPFc"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="phTPsoJop">Tighten the character spacing in a span of text in the PDF. See &#8220;<a href="{% post_url 2019-05-22-29-AdjustletterspacingPDF-only %}"><span class="Hyperlink">Adjust letter spacing (PDF-only)</span></a>&#8221; for more.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p0zwngYoS">HED SPAN Line break after</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="liHwDa4L6q"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pdOS5aeeN">Insert a line break after a word or character. See &#8220;<a href="{% post_url 2019-05-22-30-ForcealinebreakPDF-only %}"><span class="Hyperlink">Force a line break (PDF-only)</span></a>&#8221; for more.</p></dd>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pAijcEQC2">HED SPAN Page break after</dt>
    <dd class="hblkdefinition" data-hederis-type="hblkdefinition" id="li2ZrO71ZS"><p class="hblkdefinition" data-hederis-type="hblkdefinition" id="pPeNWT5eS">Insert a line break after a word or character. See &#8220;<a href="{% post_url 2019-05-22-31-ForceapagebreakPDF-only %}"><span class="Hyperlink">Force a page break (PDF-only)</span></a>&#8221; for more.</p></dd>
    <dd/></dl>

    <section class="hwprsubsection" data-hederis-type="hwprsubsection" id="pE2cO2ASa" data-type="subsection" title="Wrappers"><h1 data-hederis-type="hblktitle" class="hblktitle" id="pwGjHYSB6">Wrappers</h1>
    <dl class="hwprdef-list" data-hederis-type="hwprdef-list" id="pih0zgc1O"><dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pdM6U9gKO">HED BOX Box start, HED BOX Box end</dt>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="pQK3OMn1m">HED BOX Bullet List start, HED BOX Bullet List end</dt>
    <dt data-hederis-type="hblkdefterm" class="hblkdefterm" id="p51wdiJRr">HED BOX Code Block start, HED BOX Code Block end</dt>
