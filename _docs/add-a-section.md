class="hblkoli" data-hederis-type="hblklip" id="pvi9VdOWm">Open the Styles pane (see &#8220;<a href="{% link _docs/fine-tune-styles.md %}" data-hederis-type="hspana" id="p9NGSd5Yd"><span
