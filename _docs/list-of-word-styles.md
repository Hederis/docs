Also often used as a heading preceding a letter body, e.g., &#8220;CONFIDENTIAL&#8221;.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pbBQ3uvAz">HED Generic subheading</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="paISZQOyr">Similar to the generic heading, a second level of heading that can be used when you want to create a logical grouping of text without creating a formal sub-section.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pPL7hdBV7">HED Image holder</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pFEIiUFfH">A placeholder for an image. See &#8220;<a href="#AddanImage" class="hspana" data-hederis-type="hspana" id="pBrkzFv7w">Add an image</a>&#8221; for more.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pFhiAU8rP">HED Image source</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="ptWwhNDTy">The source of an image, e.g. &#8220;Courtesy of Getty Images&#8221;.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="ppp8Y0bau">HED Imprint name</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pSel2tB8C">The name of your imprint, often included on a titlepage.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pKNNW5uJX">HED Line space</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pwhPvaPSh">A line space, often used to create a logical break in time or context.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="ptUEr3iku">HED List item (contd)</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pu2KOxEMQ">A second paragraph in a multi-paragraph list item (can be used for both bulleted and numbered list items).</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pfBrgrff2">HED List item &#8211; Bulleted</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pf0ZFSHZV">A bulleted or unordered list item.
