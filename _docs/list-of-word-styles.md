See &#8220;<a href="#AddanImage" class="hspana" data-hederis-type="hspana" id="px9y5z9w0">Add an image</a>&#8221; for more.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pZwdCVpXn">HED Image source</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="psUmxurMo">The source of an image, e.g. &#8220;Courtesy of Getty Images&#8221;.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pKz3KaKdW">HED Imprint name</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pvGUtSPxT">The name of your imprint, often included on a titlepage.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pufhHCznK">HED Line space</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p96oOE19L">A line space, often used to create a logical break in time or context.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pCKZ5JPGR">HED List item (contd)</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pFbCqSsgP">A second paragraph in a multi-paragraph list item (can be used for both bulleted and numbered list items).</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="p01pskzYQ">HED List item &#8211; Bulleted</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pW7KiuC9v">A bulleted or unordered list item.
