This paragraph is an example of the definition paragraph that corresponds to the term above.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="paZXvNxXM">HED Definition definition (contd)</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pFek9sWCp">A continuation paragraph for a multi-paragraph definition, as defined above.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pz4X4E5n3">HED Definition term</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p84hHncuh">A term to be defined, often followed by a <em data-hederis-type="hspanem" id="pTAJgtGpL">HED Definition definition</em> paragraph.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pTvdEillN">HED Dialogue paragraph</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pMJaaZ25Y">Dialogue.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pJxmHWjKg">HED Endnote paragraph</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p8Bn86w96">A paragraph in an endnote.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pnKnV0Mp0">HED Footnote paragraph</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="ptmJDF2Uk">A paragraph in a footnote.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="p5FMnhdZx">HED Generic heading</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pgT9mRVZ4">A generic heading, often used in the middle of boxes or sidebars when you want to introduce a block of text without creating a formal sub-section. Also often used as a heading preceding a letter body, e.g., &#8220;CONFIDENTIAL&#8221;.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pEyMn4YBJ">HED Generic subheading</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pGKp6QA9w">Similar to the generic heading, a second level of heading that can be used when you want to create a logical grouping of text without creating a formal sub-section.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="px1Pu9u13">HED Image holder</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pbvfonwA9">A placeholder for an image. See &#8220;<a href="#AddanImage" class="hspana" data-hederis-type="hspana" id="p3XOJe8S2">Add an image</a>&#8221; for more.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pbtiBZ8Sr">HED Image source</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="ppk9IBwsm">The source of an image, e.g. &#8220;Courtesy of Getty Images&#8221;.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pM6mQUYqv">HED Imprint name</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pwtwyrA6i">The name of your imprint, often included on a titlepage.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pIIxsqn5m">HED Line space</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="puwFyQjGm">A line space, often used to create a logical break in time or context.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="psEvJTKjl">HED List item (contd)</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pnZj83J10">A second paragraph in a multi-paragraph list item (can be used for both bulleted and numbered list items).</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pmfFc5sXI">HED List item &#8211; Bulleted</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pzaaq8QJs">A bulleted or unordered list item.
