See &#8220;<a href="#AddanImage" class="hspana" data-hederis-type="hspana" id="pExvERH29">Add an image</a>&#8221; for more.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pRbfWTe6w">HED Image source</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pWgyd4o6n">The source of an image, e.g. &#8220;Courtesy of Getty Images&#8221;.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pXioNmU8b">HED Imprint name</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p0DnI5mio">The name of your imprint, often included on a titlepage.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pZaLaGWTh">HED Line space</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pyuat78OH">A line space, often used to create a logical break in time or context.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pO8vlrc8F">HED List item (contd)</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pRHQzprG1">A second paragraph in a multi-paragraph list item (can be used for both bulleted and numbered list items).</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="poHw5C1Mj">HED List item &#8211; Bulleted</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="peJiqv8hQ">A bulleted or unordered list item.
