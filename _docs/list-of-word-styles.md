See &#8220;<a href="#AddanImage" class="hspana" data-hederis-type="hspana" id="p1GE4qv7f">Add an image</a>&#8221; for more.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pSuFeqFfO">HED Image source</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pCp0egAAR">The source of an image, e.g. &#8220;Courtesy of Getty Images&#8221;.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pw1fcgOy6">HED Imprint name</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="ptcKOcNCI">The name of your imprint, often included on a titlepage.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pcmz7VOuW">HED Line space</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p2kmlD8RJ">A line space, often used to create a logical break in time or context.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pOQ0MauoV">HED List item (contd)</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="puvUcte8p">A second paragraph in a multi-paragraph list item (can be used for both bulleted and numbered list items).</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pVlpj0YzC">HED List item &#8211; Bulleted</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p2cyJiYF1">A bulleted or unordered list item.
