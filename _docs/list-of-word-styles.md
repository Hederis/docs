Also often used as a heading preceding a letter body, e.g., &#8220;CONFIDENTIAL&#8221;.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="perXxCqtc">HED Generic subheading</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pVe8CbKAv">Similar to the generic heading, a second level of heading that can be used when you want to create a logical grouping of text without creating a formal sub-section.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pE1qr2w2t">HED Image holder</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p26mN3SnJ">A placeholder for an image. See &#8220;<a href="#AddanImage" class="hspana" data-hederis-type="hspana" id="pD3A3XkF4">Add an image</a>&#8221; for more.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pl47Hk1VC">HED Image source</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="plPRwQo8W">The source of an image, e.g. &#8220;Courtesy of Getty Images&#8221;.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pL2CfYrIV">HED Imprint name</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pE6YfR5C8">The name of your imprint, often included on a titlepage.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="p9q5e29yC">HED Line space</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pgzZ1Xn7Z">A line space, often used to create a logical break in time or context.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pXTekgl02">HED List item (contd)</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pH2FUz1pe">A second paragraph in a multi-paragraph list item (can be used for both bulleted and numbered list items).</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="prkjRmIFs">HED List item &#8211; Bulleted</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pAdY4V9k8">A bulleted or unordered list item.
