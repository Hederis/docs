This paragraph is an example of the definition paragraph that corresponds to the term above.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pP8BLXUVl">HED Definition definition (contd)</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="ppafPtKGK">A continuation paragraph for a multi-paragraph definition, as defined above.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="p9lsCRkkR">HED Definition term</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pLMk29WUF">A term to be defined, often followed by a <em data-hederis-type="hspanem" id="pIOX9CaQv">HED Definition definition</em> paragraph.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pSZSivxmq">HED Dialogue paragraph</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pF0mlIx9p">Dialogue.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pG7ii3Wuz">HED Endnote paragraph</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pFUbinNnD">A paragraph in an endnote.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pKfMPtGOZ">HED Footnote paragraph</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pX6K75ljx">A paragraph in a footnote.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="paQcrq93E">HED Generic heading</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="po8TOUs0Z">A generic heading, often used in the middle of boxes or sidebars when you want to introduce a block of text without creating a formal sub-section. Also often used as a heading preceding a letter body, e.g., &#8220;CONFIDENTIAL&#8221;.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pm8jqMzar">HED Generic subheading</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pFuBS6r67">Similar to the generic heading, a second level of heading that can be used when you want to create a logical grouping of text without creating a formal sub-section.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pCEEMJpFb">HED Image holder</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="p7ELGKq5j">A placeholder for an image. See &#8220;<a href="#AddanImage" class="hspana" data-hederis-type="hspana" id="pCmGWe2Ed">Add an image</a>&#8221; for more.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pXdYHg0rX">HED Image source</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pRtO4scrn">The source of an image, e.g. &#8220;Courtesy of Getty Images&#8221;.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pns2pm1jl">HED Imprint name</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pocyaudv8">The name of your imprint, often included on a titlepage.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pV1jRtj76">HED Line space</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="pOxfEUgYy">A line space, often used to create a logical break in time or context.</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="p4B0m2O1j">HED List item (contd)</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="ph6O5VX4q">A second paragraph in a multi-paragraph list item (can be used for both bulleted and numbered list items).</p><p data-hederis-type="hblkdefterm" class="hblkdefterm" id="pAiBBtnyE">HED List item &#8211; Bulleted</p><p class="hblkdefinition" data-hederis-type="hblkdefinitionp" id="phsp7hpZx">A bulleted or unordered list item.
