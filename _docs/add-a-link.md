<html><body><section data-type="chapter" class="hsecchapter" data-hederis-type="hsecchapter" id="add-a-link" data-pi-attrs="id: add-a-link; data-tags: typeset;" role="doc-chapter" data-tags="typeset" data-author-name=" " data-book-title=" " title="Add a Hyperlink"><p class="hblkp" data-hederis-type="hblkp" id="pcAUgstt4">A link is a span of text that, when clicked, takes the reader to a specific webpage. There are two ways to add a link in the Hederis Edit Text pane. If you want the text of the link to be the same as the webpage url that you are linking to, and the url starts with &#8220;http&#8221; or &#8220;www&#8221;, then you can do the following:</p><ol class="hwprnumlist" data-hederis-type="hwprnumlist" id="pXuv2mepi"><li class="hblkoli" data-hederis-type="hblkoli" id="liJtuFNBxA"><p class="hblkoli" data-hederis-type="hblklip" id="pgb2PdEhQ">Open the Edit Text pane, and find the location where you want to add a link.</p></li><li class="hblkoli" data-hederis-type="hblkoli" id="li7eazBNFM"><p class="hblkoli" data-hederis-type="hblklip" id="p6GSfBZsb">If the text that you want to turn into a link does not exist yet, then add it: click to select the paragraph, and then click the pencil icon to enter text edit mode, and type your new text.</p></li><li class="hblkoli" data-hederis-type="hblkoli" id="liD3JDJkm6"><p class="hblkoli" data-hederis-type="hblklip" id="pGewwgyPt">Still in text edit mode, select the text that you want to turn into a link by dragging and highlighting it with your mouse. </p></li><li class="hblkoli" data-hederis-type="hblkoli" id="liqTURmNTx"><p class="hblkoli" data-hederis-type="hblklip" id="p3Hbo2O1c">Click the Add Inline button and choose &#8220;Inline Link&#8221;.</p><img data-hederis-type="hblkimg" class="hblkimg" id="p2waEetEd" src="/images/insertlink1.png" data-img-src="/images/insertlink1.png"/></li><li class="hblkoli" data-hederis-type="hblkoli" id="liXde1Sldq"><p class="hblkoli" data-hederis-type="hblklip" id="pEEML8zcO">Click outside the paragraph to leave text edit mode, and then click Save.</p></li></ol><p class="hblkp" data-hederis-type="hblkp" id="pBKSpuOeL">Your link will automatically be created using the text you selected as the website to link to.</p><p class="hblkp" data-hederis-type="hblkp" id="plk8f19vn">If you want the text of your link to say something different than the website url&#8212;for example, you want to link to www.google.com, but you want the link to say &#8220;Click Here&#8221;, then do the following:</p><ol class="hwprnumlist" data-hederis-type="hwprnumlist" id="pQUsiMWLS"><li class="hblkoli" data-hederis-type="hblkoli" id="lizqfHkgqJ"><p class="hblkoli" data-hederis-type="hblklip" id="pjPOYeZ0I">Open the Edit Text pane, and find the location where you want to add a link.</p></li><li class="hblkoli" data-hederis-type="hblkoli" id="liCzT7wmqc"><p class="hblkoli" data-hederis-type="hblklip" id="pY5GTHdHN">If the text that you want to turn into a link does not exist yet, then add it: click to select the paragraph, and then click the pencil icon to enter text edit mode, and type your new text.</p></li><li class="hblkoli" data-hederis-type="hblkoli" id="lid8bXV9EA"><p class="hblkoli" data-hederis-type="hblklip" id="pVrbHQhr7">You&#8217;ll need to format the link text a bit differently in this case. Before your text, type &#8220;DEST=&#8221;, then type the url you want to link to, then type &#8220;+TEXT=&#8221;, followed by the text that you want to show up inside the link. For example, if you want your link to read &#8220;Click Here&#8221; and the link to point to www.google.com, then your full string will read &#8220;DEST=www.google.com+TEXT=Click Here&#8221;.</p></li><li class="hblkoli" data-hederis-type="hblkoli" id="liAuX4pOSp"><p class="hblkoli" data-hederis-type="hblklip" id="pUUX9WUKe">Still in text edit mode, select the full string of text by dragging and highlighting it with your mouse. </p></li><li class="hblkoli" data-hederis-type="hblkoli" id="liVzXVvgaR"><p class="hblkoli" data-hederis-type="hblklip" id="pVYkGUCty">Click the Add Inline button and choose &#8220;Inline Link&#8221;.</p><img data-hederis-type="hblkimg" class="hblkimg" id="plNS6Zwta" src="/images/insertlink2.png" data-img-src="/images/insertlink2.png"/></li><li class="hblkoli" data-hederis-type="hblkoli" id="lifvKtTxek"><p class="hblkoli" data-hederis-type="hblklip" id="pSYQoNA6E">Click outside the paragraph to leave text edit mode, and then click Save.</p></li></ol><p class="hblkp" data-hederis-type="hblkp" id="ppRys1EwF">When you open the chapter in the Design preview, you&#8217;ll see your chosen text converted to a proper link. </p><div class="hwprbox box" data-hederis-type="hwprbox" id="pVpcOZR4E" data-type="sidebar"><p class="hblktype" data-hederis-type="hblktype" id="pVUYU99e9">Note</p><p class="hblkp" data-hederis-type="hblkp" id="phntVDI0s">Links are disabled in the Hederis preview, so if you try to click your new link, it won&#8217;t do anything. You can make sure it works by rebuilding and downloading the PDF or EPUB file.</p></div></section></body></html>
